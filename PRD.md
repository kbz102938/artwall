# ArtWall 产品需求文档（PRD）

## 一、产品概述

### 1.1 产品名称
ArtWall（艺墙）

### 1.2 一句话描述
基于用户审美偏好，智能推荐艺术画作，并提供实体画作交付服务。

### 1.3 核心价值
> **"根据用户审美偏好，推荐真正适合他的画作，并交付实体画作到家"**

解决用户痛点：
- 画太多不知道选什么 → AI 根据偏好帮你选
- 不懂艺术怕选错 → 基于你的审美匹配，不会出错
- 选完还要自己找地方买 → 一站式交付实体画作

效果渲染是辅助功能：帮用户确认"这幅画挂我家好不好看"，降低决策门槛

### 1.4 目标用户
- 新房装修完需要挂画的人
- 想更换家中装饰画的人
- 对艺术画感兴趣但不知如何选择的人

## 三、用户输入（Input）

### 3.1 房间照片（必填）

| 字段 | 说明 |
|-----|------|
| 数量 | 1-3张 |
| 要求 | 正面拍摄要挂画的墙面，光线清晰 |
| 格式 | JPG/PNG，建议 1MB 以上 |

### 3.2 家居风格选择（必填）

用户从预设的风格图片中选择最接近自己家的 1-2 张。

**风格选项（共8种）：**

| 编号 | 风格名称 | 关键词 |
|-----|---------|--------|
| 1 | 现代简约 | 黑白灰、线条感、少即是多 |
| 2 | 北欧 | 白色、木质、绿植、温暖 |
| 3 | 日式/侘寂 | 原木、低饱和、自然、禅意 |
| 4 | 新中式 | 木质、对称、留白、东方元素 |
| 5 | 法式/轻奢 | 石膏线、金色点缀、优雅 |
| 6 | 美式 | 深色木质、复古、厚重 |
| 7 | 工业风 | 水泥、金属、裸露管道 |
| 8 | 奶油风/混搭 | 柔和色调、舒适、无明显风格 |

### 3.3 画作偏好选择（必填）

**方式：二选一连续选择，共5轮**

每轮展示两张风格差异明显的画作，用户点选更喜欢的一张。

**选择维度覆盖：**
- 具象 vs 抽象
- 暖色 vs 冷色
- 繁复 vs 极简
- 东方 vs 西方
- 人物 vs 风景/静物

**偏好输出标签：**
- 流派偏好（印象派/古典/抽象/东方...）
- 色调偏好（暖/冷/中性/高饱和/低饱和）
- 主题偏好（风景/人物/静物/抽象）

## 四、系统处理

### 4.1 AI 分析模块

**输入：** 房间照片 + 风格选择 + 画作偏好选择结果

**输出：** 匹配分析报告（文字）

```
示例输出：
────────────────────────────────────
【空间分析】
您的空间呈现现代简约风格，以白色和浅灰色为主色调，
家具线条简洁，自然光充足，整体氛围干净利落。

【画作偏好】
您偏好冷色调、抽象或极简风格的作品，
对繁复的古典写实风格兴趣较低。

【推荐方向】
建议选择：抽象几何、极简线条、冷色调风景类作品
推荐艺术家：蒙德里安、康定斯基、莫兰迪、吴冠中
────────────────────────────────────
```

### 4.2 画库匹配模块

**匹配逻辑：**
1. 根据家居风格筛选「适合风格」标签匹配的画
2. 根据画作偏好筛选「流派」「色调」「主题」匹配的画
3. 综合打分排序，取 Top 3-5

### 4.3 效果渲染模块

**功能：** 将推荐的画作合成到用户上传的房间照片中

**技术要点：**
- 墙面区域检测
- 透视变换（让画贴合墙面角度）
- 光影融合（让画看起来自然）
- 支持切换不同候选画预览

## 五、用户输出（Output）

### 5.1 匹配分析报告

| 内容 | 说明 |
|-----|------|
| 空间风格总结 | AI 对用户家居风格的描述 |
| 画作偏好总结 | AI 对用户审美偏好的描述 |
| 推荐方向 | 适合什么类型的画、推荐艺术家 |

### 5.2 候选画作推荐

每幅画包含：

| 字段 | 说明 |
|-----|------|
| 画作名称 | 如《星月夜》 |
| 作者 | 如 文森特·梵高 |
| 年代 | 如 1889 |
| 流派 | 如 后印象派 |
| 缩略图 | 画作预览图 |
| 推荐理由 | 为什么这幅画适合用户 |

### 5.3 效果渲染图

| 内容 | 说明 |
|-----|------|
| 渲染效果图 | 画挂在用户墙上的合成效果 |
| 支持切换 | 可切换不同候选画查看效果 |
| 可下载 | 用户可保存效果图 |

## 六、数据需求（待建设）

### 6.1 画库数据

**数据来源（公共版权）：**

| 来源 | 网址 | 说明 |
|-----|------|------|
| WikiArt | wikiart.org | 大量西方艺术作品 |
| 大都会博物馆 | metmuseum.org | Open Access 藏品 |
| 芝加哥艺术博物馆 | artic.edu | CC0 公共领域作品 |
| 国家美术馆(US) | nga.gov | 公共领域藏品 |
| 史密森尼 | si.edu | Open Access |
| Unsplash | unsplash.com | 艺术摄影类 |

**需要爬取/整理的字段：**

| 字段 | 类型 | 说明 |
|-----|------|------|
| id | string | 唯一标识 |
| title | string | 画作名称 |
| title_cn | string | 中文名称（需翻译） |
| artist | string | 作者 |
| artist_cn | string | 作者中文名 |
| year | number | 创作年代 |
| style | string | 流派（印象派、抽象...） |
| image_url | string | 原图链接 |
| image_hd_url | string | 高清图链接 |
| source | string | 来源（WikiArt/Met...） |
| source_url | string | 原始页面链接 |
| license | string | 版权信息 |

**需要人工/AI 打标的字段：**

| 字段 | 类型 | 说明 |
|-----|------|------|
| color_tone | string[] | 色调标签：暖色/冷色/中性/高饱和/低饱和 |
| theme | string[] | 主题标签：风景/人物/静物/抽象/建筑 |
| mood | string[] | 氛围标签：宁静/活力/忧郁/浪漫/神秘 |
| suitable_styles | string[] | 适合的家居风格 |
| aspect_ratio | string | 横幅/竖幅/方形 |

**MVP 画库规模建议：**
- 初期：500-1000 幅精选画作
- 覆盖主要流派和风格
- 确保每种家居风格都有足够匹配选项

### 6.2 家居风格示例图

**需要准备：**
- 8种风格 × 每种2-3张 = 16-24张家居图片
- 要求：清晰、典型、无版权问题
- 来源：Unsplash、Pexels 等免费图库

### 6.3 偏好选择用画作

**需要准备：**
- 用于「二选一」偏好测试的画作对
- 5轮 × 每轮2张 = 10张（需精心挑选差异明显的对比）
- 覆盖各种风格维度
